// syntax

Clean. Familiar.
Powerful.

hello.kl
# Variables & output
x = 5 + 3
y = x * 2
out y

# For loop
for i = 0 to 10:
    out i

# Condition
if x > 5:
    out x
else:
    out 0
functions.kl
# Define functions
def add(a, b):
    result = a + b
    out result

add(5, 9)

# Typical iterative pattern
# Clear and maintainable by design
sum = 0
for i = 0 to 1000000000:
    sum = sum + i
out sum
input.kl
# User input
x = input()
out x

# Factorial
def factorial(n):
    result = 1
    for i = 1 to n:
        result = result * i
    out result

factorial(15)
terminal
# Run a program
$ klang run hello.kl

# Compile to native
$ klang build prog.kl -o app

# Choose backend
$ klang run --backend llvm fib.kl
$ klang run --backend vm fib.kl

# Launch IDE
$ klang ide
// features

Everything you need.
Nothing you don't.

⚡
Multiple Backends

Choose between VM bytecode and LLVM JIT/AOT. Match your performance needs to your toolchain.

🧠
Pattern Optimization

The compiler can recognize recurring patterns and apply safe optimizations automatically during compilation.

📦
Native Executables

Compile Klang programs to standalone .exe or binary files. Ship your app with no runtime dependency.

🖥️
Built-in IDE

A Tkinter-based IDE ships with Klang out of the box. Write, run, and debug without leaving the environment.

🔌
Simple Syntax

If you know Python, you already know Klang. Clean indentation-based syntax with zero boilerplate.

🚀
Extensible Roadmap

The project is structured for steady growth, with a focus on maintainability, ecosystem tooling, and long-term usability.
